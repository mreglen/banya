# Печать акта брони

## Что реализуем
Сделаем новый маршрут печатной версии для брони в формате **акта оказания услуг**: из модалки деталей брони в `/admin/reservations` будет открываться новая вкладка с готовым документом и кнопкой `Печать`, которая вызывает `window.print()`.

## Изменения по файлам
- Обновить кнопку в деталях брони: [frontend/my-banya/src/pages/Admin/Reservations/BookingDetailsModal.jsx](frontend/my-banya/src/pages/Admin/Reservations/BookingDetailsModal.jsx)
  - добавить кнопку `Печать` рядом с текущими действиями;
  - открывать новую вкладку на новый маршрут печати с `reservation_id`.
- Добавить страницу печатного документа: [frontend/my-banya/src/pages/Admin/Reservations/ReservationPrintDocument.jsx](frontend/my-banya/src/pages/Admin/Reservations/ReservationPrintDocument.jsx)
  - загрузка брони по `id` через существующий API-хук;
  - макет акта: шапка организации, клиент, данные брони, позиции, итоги, подписи;
  - верхняя панель с кнопкой `Печать` (вызов `window.print()`), скрываемая в печатном режиме;
  - печатные стили (`@media print`) для A4-подобного документа.
- Подключить маршрут: [frontend/my-banya/src/App.js](frontend/my-banya/src/App.js)
  - добавить роут вида `/admin/reservations/print/:id`.
- (При необходимости для чистоты) вынести функции расчета сумм/форматирования в helper рядом со страницей, чтобы макет не дублировал логику из модалки.

## Содержимое печатного акта (как согласовано)
- Шапка организации: название, контакты, адрес.
- Данные клиента: ФИО, телефон, email.
- Данные брони: баня, дата, время, количество гостей.
- Позиции: услуги/товары с ценами и количеством.
- Итоги: сумма, скидка/акция, итог к оплате.
- Блоки подписи: исполнитель/клиент + место для печати.

## Проверка
- Из модалки брони кнопка `Печать` открывает новый таб с документом.
- На странице печати отображаются все нужные поля.
- Кнопка `Печать` открывает системное окно печати.
- В режиме печати интерфейсные кнопки/лишние элементы не выводятся.
- Документ читаемо печатается на A4 (без обрезания ключевых блоков).
