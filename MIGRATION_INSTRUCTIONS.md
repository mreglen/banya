## Что изменилось

### URL бань
- **Было:** `/baths/1`, `/baths/2`, `/baths/3`
- **Стало:** `/baths/kedrovaya-banya`, `/baths/lipovaya-banya`, `/baths/neftovaya-banya`

### Страница бани
- Добавлены стрелки влево/вправо для навигации по фото
- Стрелки появляются только если у бани больше 1 фото
- Миниатюры теперь подсвечивают активное фото зеленой рамкой
- Улучшена навигация - не нужно переставлять фото в массиве

### Админка
- При создании бани slug генерируется автоматически из названия
- При изменении названия бани slug обновляется автоматически
- Slug всегда уникальный (при конфликте добавляется -1, -2 и т.д.)

## Примеры транслитерации

- "Кедровая баня" → `kedrovaya-banya`
- "Липовая баня" → `lipovaya-banya`
- "Нефритовая" → `neftitovaya`
- "Баня №1" → `banya-1`

## Тестирование

1. Откройте главную страницу
2. Нажмите "Подробнее" на любой бане
3. Проверьте, что URL содержит slug (например, `/baths/kedrovaya-banya`)
4. Проверьте работу стрелок на фото (если фото > 1)
5. Попробуйте создать новую баню в админке - slug сгенерируется автоматически

## Примечания

- Старые URL с ID больше не работают (используется только slug)
- Slug генерируется автоматически и уникален
- При изменении названия бани slug обновляется
- SEO meta теги обновлены для использования новых URL
