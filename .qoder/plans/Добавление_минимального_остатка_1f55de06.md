# Добавление минимального остатка для товаров

## Обзор изменений

Добавляем поле `min_stock` (минимальный остаток) для товаров, отображаем его на странице создания документа поступления, на странице номенклатуры подсвечиваем красным товары у которых остаток ниже минимального, добавляем фильтр по минимальному остатку и убираем столбец "Сайт".

---

## Задача 1: SQL скрипт для добавления колонки в БД

Создать файл `backend/add_min_stock_column.sql`:

```sql
ALTER TABLE products ADD COLUMN min_stock FLOAT DEFAULT 0.0;
```

Это добавит колонку `min_stock` со значением по умолчанию 0.0 для всех существующих товаров.
